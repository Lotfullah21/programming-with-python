Logical operators-

- Logical AND operator
- Logical OR operator
- Logical NOT operator

```table
| Operator       | Symbol | Description                                      | Example             | Result         |
|----------------|--------|--------------------------------------------------|---------------------|----------------|
| Logical AND    | `and`  | Returns `True` if both conditions are `True`     | `True and False`    | `False`        |
| Logical OR     | `or`   | Returns `True` if at least one condition is `True` | `True or False`     | `True`         |
| Logical NOT    | `not`  | Reverses the logical state of its operand         | `not True`          | `False`        |

```
